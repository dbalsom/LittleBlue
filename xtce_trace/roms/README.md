# ROMs

## 1501512.u18

This binary represents the upper 32K ROM of an IBM XT.
The 1501512.u18 file checked into this repository contains an 8K image of [GLaBIOS 0.4](https://glabios.org/), a 
clean-room, open-source reimplementation of the IBM BIOS.

You may replace it with a genuine IBM ROM at your discretion.

## 5788005.u33

This binary represents the character ROM from the IBM CGA card.
Legal precedent in the United States has determined that IBM's character ROMs are not subject to copyright protection.

https://cdn.loc.gov/copyright/history/mls/ML-393.pdf
